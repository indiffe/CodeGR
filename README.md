# CodeGR

Code-based Generative Retrieval research codebase.

## Environment
- Python 3.9 or 3.10
- PyTorch 2.0.0(CUDA 11.x)
- Transformers 4.28.0

All required Python packages are listed in `requirements.txt`.
A CUDA-enabled GPU is required for training.

## Structure
- gemkr/: core framework
- configs/: experiment configs
- scripts/: training & evaluation scripts
